The Book List

The Book List is a book listing app, created using React, linked to an API created using Express.


My new challenges in creating The Book List were:

  1. Setting up an API. I found this relatively straightforward, but it was definitely new territory.
  2. Separating the app into different components. I started to get to grips with how state and props can be used in making The Book List. I was able to put each part of the app into a different component and have components trigger methods from the main App.js file. This streamlined the process of making changes because changing one component doesn't (or at least shouldn't) affect the others.
