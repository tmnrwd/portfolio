Forecasting the Weather

Forecasting the Weather is a simple weather app, created using React. It draws its data from OpenWeather.


My new challenges in creating Forecasting the Weather were:

  1. Learning how to use API requests and how to get what I want out of their responses.
  2. The app's logic isn't very streamlined: it only has one component, which defines state, carries out the API request, does some processing on the response, and makes changes to state. This meant that any changes had to be made to the different weather cards manually, rather than creating the cards from a single place. I improved on my approach to this kind of app in later projects, using more components and linking them together to make edits easier.
